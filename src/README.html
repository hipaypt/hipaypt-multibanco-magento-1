Comprafácil - Payment Method

Unzip

Move the files inside the "files" folder in your magento website root.

Create module table

Run this query in the magento database
CREATE TABLE `comprafacil` (
`id` int(11) NOT NULL AUTO_INCREMENT,
`entity_id` int(11) NOT NULL,
`reference` text NOT NULL,
`entity` text NOT NULL,
`value` double NOT NULL,
`status` tinyint(1) DEFAULT '0',
`key` text,
PRIMARY KEY (`id`)
) ENGINE=MyISAM AUTO_INCREMENT=10 DEFAULT CHARSET=utf8;

Add multibanco payment info on the sucess file (Optional)

Usually this file is located at "app/design/frontend/base/default/template/checkout/success.phtml"
If you have a custom template you must locate this file in your template.
<?php
$order = Mage::getModel('sales/order')->loadByIncrementId($this->getOrderId());
if($order->getPayment()->getMethod() == "cf"){
$cf = Mage::getModel('cf/paymentmethod');
echo $cf->getRefhtml($this->getOrderId());
}
?>
